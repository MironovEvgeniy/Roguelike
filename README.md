Учебный проект в жанре roguelike с процедурной генерацией лабиринтов, боевой системой и «перманентной смертью». Игра построена на собственном 2D-движке поверх C++17 и SFML 2.5.1.

Геймплей
• Процедурная генерация уровней (MazeGenerator).

• Враги с ИИ (AIMovementComponent), фабрики врагов и состояния врага.

• Характеристики персонажа (StatsComponent).

• Боевая система (компоненты атак и ориентации спрайта).

• Перманентная смерть.


Движок (модуль Engine)
• Компонентно-ориентированная архитектура:

• Ввод: InputComponent

• Игровые объекты: GameObject, Component

• Рендер: RenderSystem (спрайты, камера)

• Коллизии и физика: Collision, PhysicsSystem

• Ресурсы: менеджер текстур/звуков

• Звук: SoundComponent

• Логирование

• Трансформации: позиция/поворот/масштаб

• Математика: векторы/матрицы

• Камера и игровой мир


Структура проекта
• Engine — собственный 2D-движок

• RogueLikeGame — игровая логика:

• генерация лабиринтов и создание уровней

• объекты (игрок, враги, стены, пол, музыка)

• HUD/сцены/состояния игры

• бой, ориентация спрайтов


 Технологии
• C++17

• SFML 2.5.1

• Visual Studio 2022


Установка и запуск
Вариант 1 — через Visual Studio

1.Клонировать репозиторий:

git clone https://github.com/MironovEvgeniy/Roguelike

2.Открыть Game.sln в Visual Studio 2022.

3.Назначить в качестве запускаемого проекта XYZRoguelike в обозревателе решений

4.Собрать и запустить (F5).

Вариант 2 — через бат-скрипт (Build.bat)

1.Запустить Build.bat в корне репозитория.

2.Скрипт соберёт решение, скопирует ресурсы в x64<Конфигурация>\Resources и запустит игру.

3.Исполняемый файл по умолчанию: RogueLikeGame\x64\Debug\RogueLikeGame.exe (или x64\Debug\RogueLikeGame.exe — в зависимости от конфигурации скрипта).


Управление
• WASD / стрелки — перемещение

• ЛКМ — атака

• Esc — пауза / меню
